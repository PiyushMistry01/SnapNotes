# SnapNotes

SnapNotes is a simple full-stack notes application where users can create notes with text and images. 
Each note can include a caption, date, and an uploaded image, and all notes are stored in a database and accessible via a REST API.

---

## Features

- Create notes with caption, date, and image
- Upload and store images
- View all saved notes via API
- Delete notes (if implemented)
- Simple and responsive backend design

---

## Tech Stack

- **Backend:** Node.js, Express.js
- **Database:** MongoDB (Mongoose)
- **File Uploads:** Multer (or similar middleware)
